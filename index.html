Frequently Asked Questions

What's the difference between var, let, and const?
var is function-scoped, can be redeclared and updated, and is hoisted with undefined. let is block-scoped, cannot be redeclared but can be updated, and is not hoisted with undefined. const is also block-scoped, must be initialized, and cannot be updated or redeclared.
What's the difference between map(), forEach(), and filter()?
map() creates a new array by applying a function to each element of the original array, returning the transformed values. forEach() executes a provided function on each element of the array but does not return a new array (it's used for side effects). filter() creates a new array with elements that pass the provided test function, returning only the elements that meet the condition.
